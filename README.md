# stairway.space

Personal website and academic portfolio — [stairway.space](https://stairway.space)

### About

Static personal site for Damien Koon: physics undergrad at Florida Institute of Technology (graduating 2027), focused on theoretical physics, quantum mechanics, and gravitational theory. Includes research notes, projects (e.g. CHSH inequality, gamma-ray burst afterglows), outreach work, and a simple gallery.

### Tech

- Plain **HTML** and **CSS** (no frameworks)
- Responsive layout with a space-themed starry background
- Contact form powered by [FormSubmit.co](https://formsubmit.co)
- Favicons and basic SEO (canonicals, meta, sitemap)

### Structure

| Path | Description |
|------|-------------|
| `index.html` | Home + intro and external links |
| `about.html` | Academic journey, research interests, outreach |
| `projects.html` | Research notes & projects |
| `gallery.html` | Photo gallery (masonry-style) |
| `contact.html` | Contact form |
| `changelog.html` | Site changelog |
| `css/` | Global and contact-specific styles |
| `images/` | Logos, favicons, icons |
| `gallery/` | Gallery images |
| `academic_materials/` | CV and résumé PDFs |
